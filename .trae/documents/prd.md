## 1. 产品概述

聚合型个人网站，聚焦头脑风暴协作与信息对齐场景。以暗黑风格为视觉基调，通过交互式网格背景、圆形右键菜单、翻面头像音频播放、翻页数字时钟等特色交互，打造具有记忆点的个人展示与协作入口。

- 目标用户：创意工作者、技术从业者、需要快速对齐信息的团队协作者
- 核心价值：以沉浸式交互体验替代传统静态个人主页，将信息展示与协作触发融为一体

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览内容、使用右键菜单导航、触发头像交互 |
| 站长 | 本地配置 | 管理菜单配置、替换头像/音频资源 |

### 2.2 功能模块

1. **主页面**：交互式网格背景、头像展示区、个人/项目介绍层、翻页数字时钟
2. **右键菜单系统**：圆形放射菜单、菜单项导航、配置化管理
3. **头像交互系统**：翻面动画、音频播放、果冻缩放动效

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | InteractiveGridPattern 背景 | SVG 交互式网格图案，响应式铺满视口，悬停高亮效果，不影响前景可读性 |
| 主页面 | 头像展示区（第一层） | 圆形头像+2px白色边框+背光外发光，左键翻转显示第二张图，翻面后点击播放MP3，果冻缩放动效 |
| 主页面 | 个人/项目介绍层（第二层） | 文字介绍区域，支持个人简介与项目描述展示 |
| 主页面 | 翻页数字时钟（第三层） | 翻页式数字时钟组件，实时显示当前时间，翻页动画过渡 |
| 全局 | 右键圆形菜单 | 长按右键触发，以鼠标位置为中心的圆形菜单，旋转动画300ms，悬停显示名称，点击跳转，JSON配置驱动 |

## 3. 核心流程

**访客浏览流程**：用户进入网站 → 看到暗黑风格网格背景 + 三层内容 → 可通过右键长按打开圆形菜单导航 → 可点击头像触发翻转/音频交互 → 底部时钟实时运行

**头像交互流程**：初始状态（正面头像+边框+背光）→ 左键点击 → 果冻缩放动效(200ms) + 翻转动画(500ms) → 翻面状态（背面图片，无边框）→ 再次点击 → 果冻缩放动效 + 随机播放MP3 → 支持多音频同时播放

**右键菜单流程**：长按右键 → 以鼠标位置为中心展开圆形菜单(300ms旋转动画) → 悬停菜单项显示名称tooltip → 点击菜单项跳转对应URL → 点击空白区域关闭菜单

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#1a1a2e（深蓝黑）作为背景主色，#121212 作为辅助深色
- **辅助色**：#e0e0e0（浅灰文字，对比度 12.6:1）、#64ffda（青绿强调色，对比度 9.3:1）、#ff6b6b（暖红交互反馈色）
- **按钮风格**：圆角胶囊形，悬停时发光边框效果
- **字体**：标题使用 "Playfair Display"（衬线装饰风），正文使用 "DM Sans"（现代几何无衬线）
- **布局风格**：垂直流式居中布局，大留白，极简克制
- **图标风格**：线性图标（Lucide Icons），1.5px 描边
- **动效风格**：克制优雅，关键交互点有明确反馈，避免过度动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | InteractiveGridPattern 背景 | SVG网格，stroke色 rgba(100,255,218,0.15)，悬停填充 rgba(100,255,218,0.08)，绝对定位铺满视口 |
| 主页面 | 头像展示区 | 圆形容器 160px(桌面)/120px(平板)/96px(移动)，2px白色边框，box-shadow: 0 0 20px rgba(100,255,218,0.4)，CSS 3D翻转 |
| 主页面 | 个人/项目介绍层 | 居中文本块，标题 Playfair Display 2rem，正文 DM Sans 1rem，行高1.8，最大宽度680px |
| 主页面 | 翻页数字时钟 | 翻页卡片式数字，每片高60px宽40px，深色卡片+浅色数字，翻页过渡transform-origin: bottom |
| 全局 | 右键圆形菜单 | 圆形半径120px，6-8个菜单项等角分布，图标+tooltip，展开旋转动画300ms ease-out |

### 4.3 响应式设计

- **桌面端（1024px+）**：完整布局，头像160px，菜单半径120px
- **平板端（768px-1023px）**：头像120px，菜单半径100px，字号适当缩小
- **移动端（320px-767px）**：头像96px，菜单半径80px，右键改为长按触摸触发，时钟字号缩小
- 触摸设备适配：使用 touchstart/touchend 事件模拟长按右键行为
